## AI-Powered Sales Forecasting Dashboard

An advanced machine learning sales forecasting dashboard built using Python, XGBoost, Streamlit, and Plotly.

## Project Overview
This project predicts future sales revenue using historical retail sales data and machine learning forecasting techniques.

The dashboard provides:
- Revenue forecasting
- Actual vs predicted analysis
- Feature importance analysis
- Interactive business intelligence visualizations
- Store and category performance insights

## Technologies used
- Python
- Pandas
- NumPy
- XGBoost
- Scikit-learn
- Plotly
- Streamlit
